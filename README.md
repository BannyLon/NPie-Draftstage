### 🎯 为什么需要哌稿场？

自媒体创作不是写一篇就完了——多条选题并行推进、商单和自制内容混在一起、发布日期倒逼制作节奏、周末节假日还要绕开。大多数人用 Excel 或备忘录硬撑，结果就是：**漏节点、算错日期、忘记谁先谁后**。

哌稿场做三件事：

1. **把排期可视化** — 横向日历 + 纵向选题 = 一眼看清每条内容从策划到发布的全链路
2. **把规则自动化** — 选定发布日期，倒排算法自动算出每个制作节点的截止日期，跳过周末和法定假日
3. **把风险前置** — 距发布日期越近、商单金额越大、标记越紧急，跑马灯越醒目，不会事到临头才发现还没拍

### 🏆 核心亮点

| 亮点 | 说明 |
|------|------|
| **工作日倒排引擎** | 输入发布日期 → 自动反推各阶段节点，避开周末 + 2026年中国法定节假日 |
| **双轨工作流** | 自制内容（9阶段）与商单（11阶段 + 品牌审核 + 金额标注）各自独立 |
| **自定义工作流** | 设置面板中自由定义执行流程，倒排算法自动适配 |
| **跑马灯预警** | 综合发布日期、紧急标记、星评影响力、商单金额多维度排序滚动提示 |
| **Obsidian 联动** | 每条选题绑定 `obsidian://` 笔记链接，点一下从看板跳转到创作工具 |
| **暗色模式 + 6 主题** | 日间/夜间一键切换，暖棕/冷灰/自然绿/时尚杂志/草绿清新/暖金六套配色 |
| **零依赖纯前端** | 一个 HTML + 两个文件（CSS/JS），浏览器打开即用，数据存 localStorage |
| **右键上下文感知** | 空格子 → 新增日程，有任务 → 删除节点，进度 100% → 存档，0% → 放弃 |

## ✨ 功能特性

### 📅 排期日历
- 横向时间轴（7月–12月，随选题自动扩展）
- 每个选题一行，任务块按日期摆放，拖拽即可调整排期
- **工作日倒排**：自动跳过周末与中国法定节假日（2026年）
- 封面/文案/发布三个发布日节点自动堆叠显示
- 同一日期的多个任务自动上下错开，互不遮挡

### 📋 选题卡
- 前置准备清单（自由增删、勾选）
- 制作流程自动倒排（从发布日期往前推算各阶段日期）
- 进度条实时联动（前置准备 + 流程节点统一计算）
- 选题卡背景色区分**自制内容**（暖米色）与**商单**（冷灰蓝）

### 🔄 两种工作流

| 类型 | 阶段顺序 |
|------|----------|
| 自制内容 | 前置准备 → 脚本 → A-roll → B-roll → 剪辑(2天) → 包装 → 封面/文案/发布 |
| 商单 | 收到并拆解 Brief(2天) → 完成脚本大纲(3天) → 品牌审核(2天) → 脚本 → A-roll → B-roll → 剪辑(2天) → 包装 → 封面/文案/发布 |

### 📦 选题生命周期
- **进行中**（1%–99%）：正常编辑，右键菜单不显示危险操作
- **已完成**（100%）：右键 → 存档，移入"已存档"视图
- **已放弃**（0%）：右键 → 放弃选题，确认后删除
- 已存档选题可随时恢复或彻底删除

### 🎯 其他亮点
- 侧边栏可折叠（展开/收起），收起后导入导出按钮显示为图标
- 顶部通知跑马灯：自动滚动显示紧急（≤3天）、临近（≤7天）、已完成选题
- 数据持久化到浏览器 localStorage，支持 JSON 导出/导入备份
- 右键菜单上下文感知：空格子→新增日程，有任务格子→删除节点
- 任务块双击重命名，拖拽单个节点微调日期

## 🖥️ 使用方式

### 直接使用
1. 下载整个 `111/` 文件夹
2. 用浏览器打开 `content-os.html` 即可（Chrome / Edge / Safari / Firefox）

> 需联网加载 Tailwind CSS CDN 和 Google Fonts，首次加载可能需要几秒。本地上线后可将 CDN 资源下载到本地。

### 操作指南

| 操作 | 方式 |
|------|------|
| 新建选题 | 点击日历下方「+ 增加选题」按钮，填写名称/类型/日期/状态/星评 |
| 调整排期 | **拖拽**左侧选题标签到目标日期 → 整条排期重新按工作日倒排 |
| 微调节点 | **拖拽**右侧单个任务块到目标日期 → 仅移动该节点 |
| 勾选完成 | 在选题卡中勾选前置准备项或流程节点 → 进度条自动更新 |
| 删除节点 | 右键任务块 → 删除 |
| 右键日历格 | 空格子 → 新增日程；有任务 → 删除节点 |
| 重命名 | 双击任务块文字 → 原地编辑 → Enter 确认 |
| 存档/放弃 | 选题进度 100%→右键存档；0%→右键放弃 |
| 导入/导出 | 侧边栏底部按钮 → JSON 文件备份/恢复 |
| 使用说明 | 排期日历右上角「新手指南」按钮 → 5 步引导式弹窗 |

## 🛠️ 技术栈

- **纯 HTML/CSS/JS**，三层分离，零构建工具，零框架依赖
- Tailwind CSS（CDN）提供原子类辅助
- Inter + Noto Sans SC 字体（Google Fonts）
- IndexedDB 持久化（清缓存不丢）+ localStorage 兼容 + JSON 导入/导出
- CSS 自定义属性（`--bg`, `--accent` 等）统一主题色
