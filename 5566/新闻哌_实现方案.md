# 新闻哌 - AI新闻助理实现方案

## 功能概述
每天早上自动搜索全网最新AI新闻，生成新闻简报，保存到日期文件夹，并可发送到飞书。

## 技术架构

### 1. 核心组件
- **新闻搜索模块** - 全网AI新闻抓取
- **内容总结模块** - 使用AI生成新闻简报
- **文件管理模块** - 按日期创建文件夹和文档
- **飞书集成模块** - 自动发送到飞书群组

### 2. 实现步骤

#### 第一步：新闻搜索
```python
# 使用多个新闻源
- 科技媒体网站
- AI专业博客
- 社交媒体热点
- RSS订阅源
```

#### 第二步：内容处理
```python
# AI总结流程
1. 去重和筛选
2. 重要性排序
3. 生成摘要
4. 分类整理
```

#### 第三步：文件管理
```python
# 文件夹结构
/新闻简报/
  /2024-01-15/
    - 今日AI新闻.md
    - 新闻源数据.json
  /2024-01-16/
    - 今日AI新闻.md
    - 新闻源数据.json
```

#### 第四步：飞书集成
```python
# 飞书机器人API
- 自动发送到指定群组
- 支持富文本格式
- 包含新闻链接
```

## 触发方式

### 方案一：定时任务
```bash
# 每天早上8点自动执行
0 8 * * * /path/to/news_agent.py
```

### 方案二：语音唤醒
```python
# 语音识别触发
"早上好" -> 启动新闻搜索
"新闻" -> 立即搜索
```

### 方案三：手动触发
```bash
# 命令行触发
python news_agent.py --date 2024-01-15
```

### 后端
- **Python** - 主要开发语言
