### 🆘 烂日子版本

当这一天被孩子的夜醒、工作突发事件、或自己的低能量打乱时：

```
烂日子最低标准（<30分钟总投入）：
✅ 5分钟拉伸（早上或晚上，做就行）
✅ 10分钟亲子专注陪伴（不玩手机，就10分钟）
✅ 读1页书（对，就1页）
✅ 写下明天1件事
```

**原则：做了就比没做强。0和1的差距，远大于1和100的差距。**

## 🌵 每周「留白」时间（防崩溃机制）

> 你的生活节奏密不透风：上班→带娃→学习→睡觉。没有任何无所事事的时间。
> 这恰恰是两周后崩塌的根本原因——你不是缺自律，你是缺空白。

```
每周至少保留 1 个「什么也不安排的时段」：

建议：周六下午 14:00-16:00（或你挑一个更合适的）
  - 不安排健身
  - 不安排学习
  - 不安排亲子活动（除非你想）
  - 可以躺着、瞎逛、刷手机、发呆、睡午觉

这不是浪费时间。这是你给大脑留的呼吸口。
没有这个口，前面所有计划都会在两周后被你的大脑「起义推翻」。
```

### 🆘 留白烂日子版本

```
如果连周末留白都被挤掉了：下周二晚上不学习，那30分钟变成留白。
你必须每周至少有一段「什么都不用做」的时间，否则崩塌是数学必然。
```

## 💪 健身计划（3-4次/周）

### 每周安排

| 日期 | 类型 | 内容 | 时长 |
|------|------|------|------|
| 周二 | 🏠 家庭力量 | 自重+弹力带训练 | 30-40min |
| 周四 | 🏠 家庭力量 | 自重+弹力带训练 | 30-40min |
| 周六 | 🏃 户外跑步 | 慢跑 + 快走结合 | 30-45min |
| 周日 | 🧘 可选 | 散步/拉伸/亲子户外 | 20-30min |

### 周二/周四 · 家庭力量训练（具体动作）

> 所需器材：瑜伽垫 + 弹力带（如果还没有，这周买，50块钱以内）

```
热身（5分钟）：
  - 开合跳 30秒 × 2组
  - 肩关节环绕 各10圈
  - 体前屈摸脚尖 10次
  - 高抬腿 30秒

主训练（15-20分钟，每个动作之间休息45秒）：
  —— 只做3个动作，先把习惯焊死，下个月再升级 ——
    1. 自重深蹲 12-15次 × 3组
       → 屁股往后坐，像坐椅子，膝盖不内扣
    2. 俯卧撑（跪姿） 6-10次 × 3组
       → 跪姿做不了就推墙，推墙也算
    3. 臀桥 12-15次 × 3组
       → 顶峰夹紧屁股，停2秒

  如果状态好，加一个自选动作（不强制）：
    □ 平板支撑 20-30秒 × 2组
    □ 或弹力带划船 12次 × 2组

整理拉伸（5分钟）：
  - 大腿前侧拉伸 各30秒
  - 大腿后侧拉伸 各30秒
  - 胸部拉伸 各30秒
  - 婴儿式 1分钟
```

### 周六 · 户外跑步

```
方案A（状态好）：慢跑 4-5km，配速不管，跑完就行
方案B（状态一般）：跑2分钟 + 快走1分钟，循环8-10次
方案C（状态差）：快走30分钟，听播客

→ 重点是出门，不是配速。
```

### 🆘 健身烂日子版本

```
周二/周四烂版：只做A组（3个下肢动作 + 拉伸），15分钟搞定
周六烂版：带孩子去公园散步30分钟，算完成
周日烂版：跳过，不内疚
```

## 💰 财务小目标

### 7月具体的3个动作

```
目标1：记录每一笔支出（7月全月）
  - 工具：随便一个记账App，或者微信/支付宝账单截图
  - 不求分类精准，求「每一笔都记」
  - 目标：月底知道自己钱花哪儿了

目标2：取消至少1个不用的订阅
  - 周末花20分钟检查微信/支付宝/Apple订阅
  - 找到一个「自动扣款但已经不用了」的，取消它
  - 预期发现：大概率有1-3个

目标3：存一笔「不可动」的钱
  - 7月1日发工资当天，转出 ¥______（你现在就填一个确切数字）
  - 建议：月收入的10%，如果10%让生活太紧，5%也行
  - 转到一个不绑定任何支付App的账户
  - 金额不重要，动作重要。¥100也比¥0强100倍。
```

### 财务烂日子版本

```
如果7月某天超预算了：记下来，不评判，不补偿性省钱
如果存不下20%：存5%。5%都存不了？存100块。100块都存不了？存1块。
关键是养成「存钱」这个动作，不是金额。
```

## ⚠️ 你需要警惕的「崩塌信号」

| 信号 | 应对 |
|------|------|
| 「今天太累了明天补」连续出现3天 | 切换到烂日子版本，不允许跳过 |
| 「反正这周已经废了」的想法 | 没有「废了」这回事，下一秒就可以重启 |
| 周末报复性熬夜/刷手机 | 不是意志力问题，是白天没有留白时间 |
| 对复盘产生抵触 | 那周只填「感恩」3条，其余都跳过 |
| 连续2天没做任何运动 | 第3天必须做烂日子版本，不允许第3天空白 |
