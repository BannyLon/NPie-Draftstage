> **技巧名称：**自审查与迭代循环（Grill Me / Code Review）
>
> **技巧的功能：**让Claude像严格的资深导师一样，狠批自己的输出或你的方案，找出问题、漏洞和改进点，然后自动迭代优化。可反复进行，直到质量达标。
>
> **应用场景：**

- 检查简历、求职信、重要邮件

- 审阅自媒体文章或朋友圈文案

- 优化个人计划书、投标材料

- 复盘自己的学习笔记或工作总结

> **实操案例（普通人痛点：写简历总是被HR打回，找不到问题在哪里）：**

> <span class="mark">这是我刚写好的求职简历。请你作为10年HR和资深面试官，用Grill me模式狠狠审查它：找出所有问题、逻辑漏洞、表达不足、缺少的亮点和潜在风险，然后提出具体修改建议。修改后给我新版本，并解释你改了哪些地方。只有通过你的严格审查才算完成。</span>

6.  **每周生活/工作自动化规划师（Weekly Planner + Review Loop）**

> **技巧名称：**每周自动化规划师（/Goal Weekly Planning + Review Loop）
>
> **技巧的功能：**
>

>
> **应用场景：**

- 周末不想花时间规划下一周，却总是手忙脚乱

- 职场人需要平衡KPI与个人生活

- 学生/ freelancer管理学习+副业+休息

- 改善睡眠、饮食、运动等生活质量
