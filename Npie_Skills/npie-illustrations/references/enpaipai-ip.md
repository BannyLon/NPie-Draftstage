# 嗯哌哌 IP

## 角色定义

嗯哌哌是 Ian 中文正文配图的固定视觉 IP，一个温暖亲切、可爱俏皮的手绘风角色。她不是吉祥物，不是贴纸，不是装饰，而是正在用姿势和表情参与内容表达的温暖引导者。

嗯哌哌严格锁定手绘柔和线条、温暖色调、俏皮大眼睛的 v2 风格，保持高度人物辨识度。所有视觉呈现均使用干净纯白背景。

## 外形

- 长直深色中分长发，自然流畅地垂落肩部，带有柔和光泽
- 柔和的椭圆脸型，脸部轮廓圆润亲切
- 大而富有表现力的深色眼睛，眼神温暖温柔，经常带着细微眼角笑纹
- 柔和拱形的眉毛，小巧自然鼻子，粉嫩饱满的嘴唇
- 白皙光滑的皮肤，带有自然的淡淡红晕
- 标志性的宽松白色针织毛衣（可根据场景搭配简洁裤装、裙装或休闲下装）
- 整体可爱圆润、手绘质感明显：可见柔软的铅笔/墨线条，彩铅或轻水彩填充，温暖柔和的色调，略带轻微夸张的圆润可爱比例（眼睛稍大、脸部柔和）
- 可呈现半身或全身

## 性格

- 温柔可爱，充满亲和力和感染力
- 带一点俏皮和活泼，笑容自然温暖
- 友好乐观，令人感到安心与愉悦
- 有点小调皮但不夸张，适合表达日常情绪
- 整体给人轻松、亲切、值得信赖、充满正能量的感觉

## 常见职责

让嗯哌哌承担核心视觉和情感角色：

- 作为正文配图、插画的核心角色
- 用姿势和表情传达文章的核心情绪或观点
- 展示丰富的全身姿势：自然站立、坐姿放松、挥手引导、双手叉腰、比耶肯定
- 表达多样化的表情：眨眼俏皮、吐舌可爱、开心微笑、温柔注视、惊喜反应
- 参与场景模拟、情绪传达、概念演示
- 用手势引导读者视线到关键信息
- 用表情呼应文章的情绪转折

## 禁止

- 不要把嗯哌哌画成过度Q版、极端卡通化或儿童化的吉祥物（保留一定写实面部比例和真实人物辨识度）
- 不要随意改变核心面部特征（中分长直发型、眼睛形状、笑容特点、脸型、发色）
- 不要添加复杂、杂乱或与角色抢戏的背景（坚持干净白底）
- 不要让嗯哌哌只是静态站立或边缘装饰，要让她成为动作或情感的主体
- 不要给复杂服装、过多配饰或闪亮特效，保持简洁温暖的毛衣造型和手绘质感
- 不要画得过于精致商业化或失去手绘温暖纹理与柔和线条
- 不要让表情过于夸张变形，要保持自然俏皮、温柔且可爱的平衡
- 不要偏离 v2 风格（软手绘质感、温暖色调、大眼睛圆润特征、彩铅质感）
- 不要在干净白底要求下添加道具、植物、灯光或环境元素（除非画面隐喻必须）

## 判断标准

如果去掉嗯哌哌后，画面会明显失去温暖亲切感和情感温度，说明嗯哌哌起到了核心作用；反之，如果嗯哌哌只是被动装饰而没有融入姿势、表情或动作，则需要重新调整提示词，让嗯哌哌真正成为视觉和情感的主体。

如果嗯哌哌的面部特征辨识度下降（例如发型改变、眼睛变小、笑容不符），或手绘风格被破坏（线条生硬、色彩失真、失去温暖质感），也视为不符合标准。

所有生成图像应匹配已生成的核心参考风格（参见 `assets/examples/` 中的 v2 系列图片）。
